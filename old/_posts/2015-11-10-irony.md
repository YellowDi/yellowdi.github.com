title: 去除微博上烦人的天猫广告
---
虽然不怎么刷微博，但难得去看看却是满屏的广告确实让人非常的厌烦。
所以做了这款简单的  Chrome  插件，可以有效屏蔽微博上烦人的广告以及一些很愚蠢的功能，比如：好友最近关注了谁。

```css
.WB_ad_tm2015,
/*简直是无限不要脸天猫横幅广告*/

#v6_TrustPagelet_Recom_MusicRank,
/*亚洲新歌榜实时趋势，尼玛，也不问问老子想不想看*/

#v6_pl_rightmod_recominfo,
/*热门话题，其实就是八卦的五体投地，别的啥好玩的都没有*/

#v6_trustPagelet_recom_member,
/*会员专区，推荐你关注会员，呵呵，我就不看，所以会员白花钱*/

#v6_pl_rightmod_noticeboard,
/*公告栏，然而，有什么卵用？*/

#v6_pl_rightmod_attfeed,
/*好友关注动态，我没什么兴趣，你喜欢看可以删除这一行*/

#v6_pl_rightmod_ads36,
/*右侧的广告，什么是热门商品推荐*/

#v6_pl_ad_bottomtip,
/*页尾的广告，好大一条*/

#v6_pl_content_biztips,
/*发布框底下那个横幅广告*/

#WB_webim,
/*右下角的私信聊天，我不用，你要是觉得有用就删除这一行*/

.key,
/*发布框右上角的各种微博推送，看着恶心*/

.WB_main_r > div:nth-child(4),
/*别问我这是啥玩意了，我记不清，连个类都没有，挺奇葩的*/

.WB_cardwrap[feedtype="ad"],
/*信息流里面的广告，汤锅里的老鼠屎*/

.WB_cardwrap[node-type="feed_spread"]
/*好友关注，我关注不关注有你屁事，自作多情*/

{
  display: none;
}
#skin_cover_s {
  background: none !important;
}
/*这是右上角头像处的背景图，曾经被放了一只天猫，怒而删之，你也可以去掉这一行*/

.WB_ad_tm2015_v2 {
  opacity: 0;
}
/*天猫侧边栏推荐商品，你一定在逗我*/
```
